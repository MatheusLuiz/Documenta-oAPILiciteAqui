Listar

Não há necessidade de usar parâmetros para utilização desta API

Cadastrar

Para cadastrar um novo registro na tabela Login Usuários, envie uma solicitação contendo os seguintes parâmetros:

usuario: Um parâmetro do tipo int, ID único do usuário (campo obrigatório).

username: Um parâmetro do tipo string, Nome de usuário para login (campo obrigatório).

senha: Um parâmetro do tipo string, Senha do usuário para acesso ao sistema (campo obrigatório).

usuario_log: Um parâmetro do tipo int, ID do usuário responsável pela criação do cadastro (campo obrigatório).

Atualizar

Para atualizar um registro existente na tabela Login Usuários, envie uma solicitação contendo os seguintes parâmetros:

id_login: Um parâmetro do tipo int, ID único do login que será atualizado (campo obrigatório).

username: Um parâmetro do tipo string, Nome de usuário para login (campo obrigatório).

senha: Um parâmetro do tipo string, Nova senha do usuário para acesso ao sistema.

usuario_log: Um parâmetro do tipo int, ID do usuário responsável pela atualização do cadastro (campo obrigatório).

Deletar

Para deletar um registro existente na tabela Login Usuários, envie uma solicitação contendo os seguintes parâmetros:

id_login: Um parâmetro do tipo int, ID único do login que será deletado (campo obrigatório).

usuario_log: Um parâmetro do tipo string, ID do usuário responsável pela exclusão do registro (campo obrigatório).
