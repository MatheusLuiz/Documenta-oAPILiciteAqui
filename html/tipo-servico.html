Listar Tipos de Serviço

Para listar todos os tipos de serviço disponíveis, utilize o seguinte endpoint:

  • GET /listar: Lista todos os tipos de serviço.

Cadastrar Tipo de Serviço

Para cadastrar um novo tipo de serviço, envie uma solicitação POST contendo os seguintes parâmetros no corpo da requisição:

  • descricao: String - Descrição do tipo de serviço (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pelo cadastro (campo obrigatório).

Atualizar Tipo de Serviço

Para atualizar um registro existente na tabela Tipo Serviço, envie uma solicitação PUT contendo os seguintes parâmetros no corpo da requisição:

  • id_tipo_servico: Inteiro - ID único do tipo de serviço a ser atualizado (campo obrigatório).
  • descricao: String - Nova descrição do tipo de serviço (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pela atualização (campo obrigatório).

Deletar Tipo de Serviço

Para deletar um registro existente na tabela Tipo Serviço, envie uma solicitação DELETE contendo os seguintes parâmetros no corpo da requisição:

  • id_tipo_servico: Inteiro - ID único do tipo de serviço a ser deletado (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pela exclusão (campo obrigatório).
