Listar

Não há necessidade de usar parâmetros para utilização desta API

Cadastrar

Para cadastrar um novo registro na tabela Tipo Serviço, envie uma solicitação contendo os seguintes parâmetros:

descricao: Um parâmetro do tipo string, Descrição do tipo de serviço (campo obrigatório).

usuario: Um parâmetro do tipo int, ID do usuário responsável pelo cadastro (campo obrigatório).

Atualizar

Para atualizar um registro existente na tabela Tipo Serviço, envie uma solicitação contendo os seguintes parâmetros:

id_tipo_servico: Um parâmetro do tipo int, ID único do tipo de serviço a ser atualizado (campo obrigatório).

descricao: Um parâmetro do tipo string, Nova descrição do tipo de serviço (campo obrigatório).

usuario: Um parâmetro do tipo int, ID do usuário responsável pela atualização (campo obrigatório).

Deletar

Para deletar um registro existente na tabela Tipo Serviço, envie uma solicitação contendo os seguintes parâmetros:

id_tipo_servico: Um parâmetro do tipo int, ID único do tipo de serviço a ser deletado (campo obrigatório).

usuario: Um parâmetro do tipo int, ID do usuário responsável pela exclusão (campo obrigatório).
