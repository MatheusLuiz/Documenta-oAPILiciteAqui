Listar Tipos de Telefone

Para listar todos os tipos de telefone disponíveis, utilize o seguinte endpoint:

  • GET /listar: Lista todos os tipos de telefone.

Cadastrar Tipo de Telefone

Para cadastrar um novo tipo de telefone, envie uma solicitação POST contendo os seguintes parâmetros no corpo da requisição:

  • descricao: String - Descrição do tipo de telefone a ser cadastrado (ex.: "Residencial", "Celular") (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pela criação do registro (campo obrigatório).

Atualizar Tipo de Telefone

Para atualizar um registro existente na tabela Tipo Telefone, envie uma solicitação PUT contendo os seguintes parâmetros no corpo da requisição:

  • id_tipo_telefone: Inteiro - ID único do tipo de telefone a ser atualizado (campo obrigatório).
  • descricao: String - Nova descrição do tipo de telefone (ex.: "Residencial", "Celular") (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pela atualização do registro (campo obrigatório).

Deletar Tipo de Telefone

Para deletar um registro existente na tabela Tipo Telefone, envie uma solicitação DELETE contendo os seguintes parâmetros no corpo da requisição:

  • id_tipo_telefone: Inteiro - ID único do tipo de telefone a ser deletado (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pela deleção do registro (campo obrigatório).
