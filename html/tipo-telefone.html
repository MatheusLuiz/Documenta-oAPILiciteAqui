Listar

Não há necessidade de usar parâmetros para utilização desta API

Cadastrar

Para cadastrar um novo registro na tabela Tipo Telefone, envie uma solicitação contendo os seguintes parâmetros:

descricao: Um parâmetro do tipo string, Descrição do tipo de telefone a ser cadastrado. Exemplo: "Residencial", "Celular", etc. (campo obrigatório).

usuario: Um parâmetro do tipo int, ID do usuário responsável pela criação do registro (campo obrigatório).

Atualizar

Para atualizar um registro existente na tabela Tipo Telefone, envie uma solicitação contendo os seguintes parâmetros:

id_tipo_telefone: Um parâmetro do tipo int, ID único do tipo de telefone a ser atualizado. Este campo é obrigatório para identificar o registro que será modificado.

descricao: Um parâmetro do tipo string, Nova descrição do tipo de telefone. Exemplo: "Residencial", "Celular", etc. (campo obrigatório).

usuario: Um parâmetro do tipo int, ID do usuário responsável pela atualização do registro (campo obrigatório).

Deletar

Para deletar um registro existente na tabela Tipo Telefone, envie uma solicitação contendo os seguintes parâmetros:

id_tipo_telefone: Um parâmetro do tipo int, ID único do tipo de telefone a ser deletado. Este campo é obrigatório para identificar o registro a ser removido.

usuario: Um parâmetro do tipo int, ID do usuário responsável pela deleção do registro (campo obrigatório).
