Listar Licitações

Para listar todas as licitações disponíveis, utilize o seguinte endpoint:

  • GET /listar: Lista todas as licitações.

Cadastrar Licitação

Para cadastrar uma nova licitação, envie uma solicitação POST contendo os seguintes parâmetros no corpo da requisição:

  • num_licitacao: String - Número único da licitação (campo obrigatório).
  • modalidade: Inteiro - Modalidade da licitação (ex.: 1 para Pregão, 2 para Concorrência, etc.) (campo obrigatório).
  • orgao: String - Órgão responsável pela licitação (ex.: Prefeitura Municipal) (campo obrigatório).
  • portal: String - Portal onde a licitação foi publicada (ex.: ComprasNet) (campo obrigatório).
  • numero_identificacao: String - Número de identificação da licitação (ex.: 12345/2024) (campo obrigatório).
  • status_licitacao: Inteiro - Status atual da licitação (ex.: 1 para Aberta, 2 para Finalizada, etc.) (campo obrigatório).
  • objeto: String - Descrição do objeto da licitação (ex.: Aquisição de materiais escolares) (campo obrigatório).
  • cidade: String - Cidade onde a licitação será realizada (ex.: São Paulo) (campo obrigatório).
  • estado: String - Estado onde a licitação será realizada (ex.: SP) (campo obrigatório).
  • data_licitacao: String - Data prevista para a licitação no formato ISO (YYYY-MM-DD) (campo obrigatório).
  • usuario: Inteiro - ID do usuário que está cadastrando a licitação (campo obrigatório).

Atualizar Licitação

Para atualizar um registro existente na tabela Licitações, envie uma solicitação PUT contendo os seguintes parâmetros no corpo da requisição:

  • id_licitacao: Inteiro - ID único da licitação que será atualizado (campo obrigatório).
  • num_licitacao: String - Novo número da licitação (campo obrigatório).
  • modalidade: Inteiro - Modalidade da licitação (ex.: 1 para Pregão, 2 para Concorrência, etc.) (campo obrigatório).
  • orgao: String - Órgão responsável pela licitação (campo obrigatório).
  • portal: String - Portal onde a licitação foi publicada (campo obrigatório).
  • numero_identificacao: String - Número de identificação da licitação (campo obrigatório).
  • status_licitacao: Inteiro - Status atual da licitação (campo obrigatório).
  • objeto: String - Descrição do objeto da licitação (campo obrigatório).
  • cidade: String - Cidade onde a licitação será realizada (campo obrigatório).
  • estado: String - Estado onde a licitação será realizada (campo obrigatório).
  • data_licitacao: String - Nova data prevista para a licitação no formato ISO (YYYY-MM-DD) (campo obrigatório).
  • usuario: Inteiro - ID do usuário que está atualizando a licitação (campo obrigatório).

Deletar Licitação

Para deletar um registro existente na tabela Licitações, envie uma solicitação DELETE contendo os seguintes parâmetros no corpo da requisição:

  • num_licitacao: String - Número único da licitação que será deletada (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável por deletar o registro (campo obrigatório).
