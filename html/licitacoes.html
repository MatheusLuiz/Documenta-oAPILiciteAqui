Listar

Não há necessidade de usar parâmetros para utilização desta API

Cadastrar

Para cadastrar um novo registro na tabela Licitações, envie uma solicitação contendo os seguintes parâmetros:

num_licitacao: Um parâmetro do tipo string, Número único da licitação (campo obrigatório).

modalidade: Um parâmetro do tipo int, Modalidade da licitação (ex.: 1 para Pregão, 2 para Concorrência, etc.).

orgao: Um parâmetro do tipo string, Órgão responsável pela licitação (ex.: Prefeitura Municipal).

portal: Um parâmetro do tipo string, Portal onde a licitação foi publicada (ex.: ComprasNet).

numero_identificacao: Um parâmetro do tipo string, Número de identificação da licitação (ex.: 12345/2024).

status_licitacao: Um parâmetro do tipo int, Status atual da licitação (ex.: 1 para Aberta, 2 para Finalizada, etc.).

objeto: Um parâmetro do tipo string, Descrição do objeto da licitação (ex.: Aquisição de materiais escolares).

cidade: Um parâmetro do tipo string, Cidade onde a licitação será realizada (ex.: São Paulo).

estado: Um parâmetro do tipo string, Estado onde a licitação será realizada (ex.: SP).

data_licitacao: Um parâmetro do tipo string, Data prevista para a licitação (formato ISO: YYYY-MM-DD).

usuario: Um parâmetro do tipo int, ID do usuário que está cadastrando a licitação (campo obrigatório).

Atualizar

Para atualizar um registro existente na tabela Licitações, envie uma solicitação contendo os seguintes parâmetros:

id_licitacao: Um parâmetro do tipo int, ID único da licitação que será atualizado (campo obrigatório).

num_licitacao: Um parâmetro do tipo string, Novo número da licitação (campo obrigatório).

modalidade: Um parâmetro do tipo int, Modalidade da licitação (ex.: 1 para Pregão, 2 para Concorrência, 3 para Convite, etc.).

orgao: Um parâmetro do tipo string, Órgão responsável pela licitação (ex.: Prefeitura Municipal).

portal: Um parâmetro do tipo string, Portal onde a licitação foi publicada (ex.: ComprasNet).

numero_identificacao: Um parâmetro do tipo string, Número de identificação da licitação (ex.: 12345/2024).

status_licitacao: Um parâmetro do tipo int, Status atual da licitação (ex.: 1 para Aberta, 2 para Finalizada, 3 para Suspensa, etc.).

objeto: Um parâmetro do tipo string, Descrição do objeto da licitação (ex.: Aquisição de materiais escolares).

cidade: Um parâmetro do tipo string, Cidade onde a licitação será realizada (ex.: São Paulo).

estado: Um parâmetro do tipo string, Estado onde a licitação será realizada (ex.: SP).

data_licitacao: Um parâmetro do tipo string, Nova data prevista para a licitação (formato ISO: YYYY-MM-DD).

usuario: Um parâmetro do tipo int, ID do usuário que está atualizando a licitação (campo obrigatório).

Deletar

Para deletar um registro existente na tabela Licitações, envie uma solicitação contendo os seguintes parâmetros:

id_licitacao: Um parâmetro do tipo string, ID único da licitação que será deletada (campo obrigatório).

usuario: Um parâmetro do tipo int, ID do usuário responsável por deletar o registro (campo obrigatório).
