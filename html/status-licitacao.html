Listar Status de Licitação

Para listar todos os status de licitação disponíveis, utilize o seguinte endpoint:

  • GET /listar: Lista todos os status de licitações.

Cadastrar Status de Licitação

Para cadastrar um novo status de licitação, envie uma solicitação POST contendo os seguintes parâmetros no corpo da requisição:

  • nome_status: String - Nome do status de licitação a ser cadastrado (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pelo cadastro (campo obrigatório).

Atualizar Status de Licitação

Para atualizar um registro existente na tabela Status Licitação, envie uma solicitação PUT contendo os seguintes parâmetros no corpo da requisição:

  • id_status: Inteiro - ID único do status de licitação que será atualizado (campo obrigatório).
  • nome_status: String - Novo nome do status de licitação (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pela atualização (campo obrigatório).

Deletar Status de Licitação

Para deletar um registro existente na tabela Status Licitação, envie uma solicitação DELETE contendo os seguintes parâmetros no corpo da requisição:

  • id_status: Inteiro - ID único do status de licitação que será deletado (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pela exclusão (campo obrigatório).
