Listar

Para listar os dados da tabela Contato Cliente, envie uma solicitação contendo o seguinte parâmetro:

cliente_id: Um parâmetro do tipo int, ID do cliente cujos contatos devem ser listados (campo obrigatório).

Cadastrar

Para cadastrar um novo registro na tabela Contato Cliente, envie uma solicitação contendo os seguintes parâmetros:

cliente: Um parâmetro do tipo int, ID do cliente associado ao contato (campo obrigatório).

tipo_telefone: Um parâmetro do tipo int, Tipo de telefone (ex.: 1 para residencial, 2 para celular, etc.).

ddd: Um parâmetro do tipo string, Código de área (DDD) do telefone (2 dígitos).

telefone: Um parâmetro do tipo string, Número do telefone (sem formatação).

nome_completo: Um parâmetro do tipo string, Nome completo do contato.

sexo: Um parâmetro do tipo string, Sexo do contato (ex.: "Masculino" ou "Feminino").

data_nascimento: Um parâmetro do tipo string, Data de nascimento no formato YYYY-MM-DD.

cpf: Um parâmetro do tipo string, CPF do contato (11 dígitos).

status_cadastro: Um parâmetro do tipo string, Status do cadastro (ex.: "Ativo" ou "Inativo").

email: Um parâmetro do tipo string, Endereço de e-mail do contato.

usuario: Um parâmetro do tipo int, ID do usuário responsável pelo cadastro.

Atualizar

Para atualizar um registro existente na tabela Contato Cliente, envie uma solicitação contendo os seguintes parâmetros:

id_cliente: Um parâmetro do tipo int, ID único do cliente associado ao contato (campo obrigatório).

cnpj: Um parâmetro do tipo string, CNPJ do cliente (14 dígitos).

razao_social: Um parâmetro do tipo string, Razão social do cliente.

nome_fantasia: Um parâmetro do tipo string, Nome fantasia do cliente.

status: Um parâmetro do tipo int, Status do cliente (ex.: 1 para ativo, 3 para inativo).

data_cadastro: Um parâmetro do tipo string, Data de cadastro no formato ISO (YYYY-MM-DD).

id_contato: Um parâmetro do tipo int, ID único do contato que será atualizado (campo obrigatório).

nome_completo: Um parâmetro do tipo string, Nome completo do contato.

ddd: Um parâmetro do tipo string, Código de área (DDD) do telefone (2 dígitos).

telefone: Um parâmetro do tipo string, Número do telefone (sem formatação).

tipo_telefone: Um parâmetro do tipo int, Tipo de telefone (ex.: 1 para residencial, 2 para celular, etc.).

sexo: Um parâmetro do tipo string, Sexo do contato (ex.: "Masculino" ou "Feminino").

cpf: Um parâmetro do tipo string, CPF do contato (11 dígitos).

email: Um parâmetro do tipo string, Endereço de e-mail do contato.

status_cadastro: Um parâmetro do tipo string, Status do cadastro (ex.: "Ativo" ou "Inativo").

Deletar

Para deletar um registro existente na tabela Contato Cliente, envie uma solicitação contendo os seguintes parâmetros:

id_contato: Um parâmetro do tipo int, ID único do contato que será deletado (campo obrigatório).

usuario: Um parâmetro do tipo int, ID do usuário responsável pela ação de exclusão.
