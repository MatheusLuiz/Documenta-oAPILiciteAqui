Listar Contatos de Clientes

Para listar todos os contatos dos clientes, utilize o seguinte endpoint:

  • GET /listar: Lista todos os contatos de clientes.

Para listar um contato específico, utilize:

  • GET /listarPorId: Lista um contato de cliente específico. Necessita do parâmetro:
    • cliente_id: ID do cliente cujos contatos devem ser listados (campo obrigatório).

Cadastrar Contato de Cliente

Para cadastrar um novo contato de cliente, envie uma solicitação POST contendo os seguintes parâmetros no corpo da requisição:

  • cliente: ID do cliente associado ao contato (campo obrigatório).
  • tipo_telefone: Tipo de telefone (ex.: 1 para residencial, 2 para celular, etc.).
  • ddd: Código de área (DDD) do telefone (2 dígitos).
  • telefone: Número do telefone (sem formatação).
  • nome_completo: Nome completo do contato.
  • sexo: Sexo do contato (ex.: "Masculino" ou "Feminino").
  • data_nascimento: Data de nascimento no formato YYYY-MM-DD.
  • cpf: CPF do contato (11 dígitos).
  • status_cadastro: Status do cadastro (ex.: "Ativo" ou "Inativo").
  • email: Endereço de e-mail do contato.
  • usuario: ID do usuário responsável pelo cadastro.

Atualizar Contato de Cliente

Para atualizar um registro existente na tabela Contato Cliente, envie uma solicitação PUT contendo os seguintes parâmetros no corpo da requisição:

  • id_contato: ID único do contato que será atualizado (campo obrigatório).
  • cliente: ID do cliente associado ao contato.
  • tipo_telefone: Tipo de telefone (ex.: 1 para residencial, 2 para celular, etc.).
  • ddd: Código de área (DDD) do telefone (2 dígitos).
  • telefone: Número do telefone (sem formatação).
  • nome_completo: Nome completo do contato.
  • sexo: Sexo do contato (ex.: "Masculino" ou "Feminino").
  • data_nascimento: Data de nascimento no formato YYYY-MM-DD.
  • cpf: CPF do contato (11 dígitos).
  • status_cadastro: Status do cadastro (ex.: "Ativo" ou "Inativo").
  • email: Endereço de e-mail do contato.
  • usuario: ID do usuário que está realizando a atualização.

Deletar Contato de Cliente

Para deletar um registro existente na tabela Contato Cliente, envie uma solicitação DELETE contendo os seguintes parâmetros no corpo da requisição:

  • id_contato: ID único do contato que será deletado (campo obrigatório).
  • usuario: ID do usuário responsável pela ação de exclusão.
