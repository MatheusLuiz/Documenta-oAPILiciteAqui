Listar Serviços de Cliente

Para listar todos os serviços de cliente disponíveis, utilize o seguinte endpoint:

  • GET /listar: Lista todos os serviços de clientes.

Cadastrar Serviço de Cliente

Para cadastrar um novo serviço de cliente, envie uma solicitação POST contendo os seguintes parâmetros no corpo da requisição:

  • id_cliente: Inteiro - ID único do cliente associado ao serviço (campo obrigatório).
  • id_tipo_servico: Inteiro - ID do tipo de serviço oferecido ao cliente (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pelo cadastro (campo obrigatório).

Atualizar Serviço de Cliente

Para atualizar um registro existente na tabela Serviços Cliente, envie uma solicitação PUT contendo os seguintes parâmetros no corpo da requisição:

  • id_servico: Inteiro - ID único do serviço a ser atualizado (campo obrigatório).
  • id_cliente: Inteiro - ID único do cliente associado ao serviço (campo obrigatório).
  • id_tipo_servico: Inteiro - ID do tipo de serviço oferecido ao cliente (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pela atualização (campo obrigatório).

Deletar Serviço de Cliente

Para deletar um registro existente na tabela Serviços Cliente, envie uma solicitação DELETE contendo os seguintes parâmetros no corpo da requisição:

  • id_servico: Inteiro - ID único do serviço a ser deletado (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pela ação de exclusão (campo obrigatório).
