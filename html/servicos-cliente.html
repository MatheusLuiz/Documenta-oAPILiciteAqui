Listar

Não há necessidade de usar parâmetros para utilização desta API

Cadastrar

Para cadastrar um novo registro na tabela Serviços Cliente, envie uma solicitação contendo os seguintes parâmetros:

id_cliente: Um parâmetro do tipo int, ID único do cliente associado ao serviço (campo obrigatório).

id_tipo_servico: Um parâmetro do tipo int, ID do tipo de serviço oferecido ao cliente (campo obrigatório).

usuario: Um parâmetro do tipo int, ID do usuário responsável pelo cadastro (campo obrigatório).

Atualizar

Para atualizar um registro existente na tabela Serviços Cliente, envie uma solicitação contendo os seguintes parâmetros:

id_servico: Um parâmetro do tipo int, ID único do serviço a ser atualizado (campo obrigatório).

id_cliente: Um parâmetro do tipo int, ID único do cliente associado ao serviço (campo obrigatório).

id_tipo_servico: Um parâmetro do tipo int, ID do tipo de serviço oferecido ao cliente (campo obrigatório).

usuario: Um parâmetro do tipo int, ID do usuário responsável pela atualização (campo obrigatório).

Deletar

Para deletar um registro existente na tabela Serviços Cliente, envie uma solicitação contendo os seguintes parâmetros:

id_servico: Um parâmetro do tipo int, ID único do serviço a ser deletado (campo obrigatório).

usuario: Um parâmetro do tipo int, ID do usuário responsável pela ação de exclusão (campo obrigatório).
