Listar

Não há necessidade de usar parâmetros para utilização desta API

Cadastrar

Para cadastrar um Cliente, envie uma solicitação contendo os seguintes parâmetros:

cnpj: Um parâmetro do tipo string, Cnpj do cliente (14 dígitos)

razao_social: Um parâmetro do tipo string, Razão social do cliente.

nome_fantasia:Um parâmetro do tipo string, Nome fantasia do cliente.

status: Um parâmetro do tipo string, Status do cliente (1 para ativo, 0 para inativo).

data_cadastro: Um parâmetro do tipo string, Data de cadastro no formato YYYY-MM-DD.

usuario: Um parâmetro do tipo string, ID do usuário responsável pelo cadastro.

Atualizar

Para atualizar um registro existente na tabela Clientes, envie uma solicitação contendo os seguintes parâmetros:

id_cliente: Um parâmetro do tipo string, ID único do cliente que será atualizado (campo obrigatório).

cnpj: Um parâmetro do tipo string, número do CNPJ do cliente (14 dígitos).

razao_social: Um parâmetro do tipo string, Razão social do cliente.

nome_fantasia: Um parâmetro do tipo string, Nome fantasia do cliente.

status: Um parâmetro do tipo inteiro, Status do cliente (ex.: 1 para ativo, 2 para inativo).

usuario: Um parâmetro do tipo inteiro, ID do usuário que está realizando a atualização.

Deletar

Para deletar um registro da tabela Clientes, envie uma solicitação contendo os seguintes parâmetros:

id_cliente: Um parâmetro do tipo inteiro, ID único do cliente que será deletado (campo obrigatório).

usuario: Um parâmetro do tipo inteiro, ID do usuário que está realizando a exclusão.
