Listar Documentos de Licitação

Para listar todos os documentos de licitação disponíveis, utilize o seguinte endpoint:

  • GET /listar: Lista todos os documentos de licitação.

Cadastrar Documento de Licitação

Para cadastrar um novo documento de licitação, envie uma solicitação POST contendo os seguintes parâmetros no corpo da requisição:

  • id_documento: Inteiro - ID único do documento de licitação (campo obrigatório).
  • num_licitacao: String - Número de identificação da licitação (ex.: "LIC00001") (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pelo cadastro (campo obrigatório).

Atualizar Documento de Licitação

Para atualizar um registro existente na tabela Documento Licitação, envie uma solicitação PUT contendo os seguintes parâmetros no corpo da requisição:

  • id_doc_licitacao: Inteiro - ID único do documento de licitação que será atualizado (campo obrigatório).
  • id_documento: Inteiro - ID único do documento de licitação associado (campo obrigatório).
  • num_licitacao: String - Número de identificação da licitação (ex.: "LIC00001") (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pela atualização (campo obrigatório).

Deletar Documento de Licitação

Para deletar um registro existente na tabela Documento Licitação, envie uma solicitação DELETE contendo os seguintes parâmetros no corpo da requisição:

  • id_doc_licitacao: Inteiro - ID único do documento de licitação que será deletado (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pela exclusão (campo obrigatório).
