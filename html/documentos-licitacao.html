Listar

Não há necessidade de usar parâmetros para utilização desta API

Cadastrar

Para cadastrar um novo registro na tabela Documento Licitação, envie uma solicitação contendo os seguintes parâmetros:

id_documento: Um parâmetro do tipo int, ID único do documento de licitação (campo obrigatório).

num_licitacao: Um parâmetro do tipo string, Número de identificação da licitação (campo obrigatório). Exemplo: "LIC00001".

usuario: Um parâmetro do tipo string, ID do usuário responsável pelo cadastro (campo obrigatório).

Atualizar

Para atualizar um registro existente na tabela Documento Licitação, envie uma solicitação contendo os seguintes parâmetros:

id_doc_licitacao: Um parâmetro do tipo int, ID único do documento de licitação que será atualizado (campo obrigatório).

id_documento: Um parâmetro do tipo int, ID único do documento de licitação associado (campo obrigatório).

num_licitacao: Um parâmetro do tipo string, Número de identificação da licitação (campo obrigatório). Exemplo: "LIC00001".

usuario: Um parâmetro do tipo int, ID do usuário responsável pela atualização (campo obrigatório).

Deletar

Para deletar um registro existente na tabela Documento Licitação, envie uma solicitação contendo os seguintes parâmetros:

id_doc_licitacao: Um parâmetro do tipo int, ID único do documento de licitação que será deletado (campo obrigatório).

usuario: Um parâmetro do tipo int, ID do usuário responsável pela exclusão (campo obrigatório).
