Listar Usuários

Para listar todos os usuários disponíveis, utilize o seguinte endpoint:

  • GET /listar: Lista todos os usuários.

Cadastrar Usuário

Para cadastrar um novo usuário, envie uma solicitação POST contendo os seguintes parâmetros no corpo da requisição:

  • nome_completo: String - Nome completo do usuário (campo obrigatório).
  • email: String - Endereço de e-mail do usuário. Deve ser um e-mail válido (campo obrigatório).
  • sexo: String - Sexo do usuário (ex.: "Masculino" ou "Feminino") (campo obrigatório).
  • data_nascimento: String - Data de nascimento do usuário no formato ISO (YYYY-MM-DD).
  • cpf: String - CPF do usuário (11 dígitos) (campo obrigatório).
  • usuario: Inteiro - ID do usuário que está cadastrando o novo registro (campo obrigatório).

Atualizar Usuário

Para atualizar um registro existente na tabela Usuários, envie uma solicitação PUT contendo os seguintes parâmetros no corpo da requisição:

  • id_usuario: Inteiro - ID único do usuário que será atualizado (campo obrigatório).
  • nome_completo: String - Nome completo atualizado do usuário.
  • email: String - Endereço de e-mail atualizado do usuário.
  • sexo: String - Sexo do usuário (ex.: "Masculino" ou "Feminino").
  • data_nascimento: String - Data de nascimento do usuário no formato ISO (YYYY-MM-DD).
  • cpf: String - CPF atualizado do usuário (11 dígitos).
  • usuario: Inteiro - ID do usuário responsável pela atualização (campo obrigatório).

Deletar Usuário

Para deletar um registro existente na tabela Usuários, envie uma solicitação DELETE contendo os seguintes parâmetros no corpo da requisição:

  • id_usuario: Inteiro - ID único do usuário que será deletado (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pela exclusão (campo obrigatório).
