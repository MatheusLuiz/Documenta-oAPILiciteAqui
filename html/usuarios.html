Listar

Não há necessidade de usar parâmetros para utilização desta API

Cadastrar

Para cadastrar um novo registro na tabela Usuários, envie uma solicitação contendo os seguintes parâmetros:

nome_completo: Um parâmetro do tipo string, Nome completo do usuário a ser cadastrado.

email: Um parâmetro do tipo string, Endereço de e-mail do usuário. Este campo é obrigatório e deve conter um endereço de e-mail válido.

sexo: Um parâmetro do tipo string, Sexo do usuário (ex.: "Masculino" ou "Feminino").

data_nascimento: Um parâmetro do tipo string, Data de nascimento do usuário no formato ISO (YYYY-MM-DD).

cpf: Um parâmetro do tipo string, CPF do usuário (11 dígitos). Este campo é obrigatório e o CPF deve ser válido.

usuario: Um parâmetro do tipo int, ID do usuário que está cadastrando o novo registro (campo obrigatório).

Atualizar

Para atualizar um registro existente na tabela Usuários, envie uma solicitação contendo os seguintes parâmetros:

id_usuario: Um parâmetro do tipo int, ID único do usuário que será atualizado (campo obrigatório).

nome_completo: Um parâmetro do tipo string, Nome completo atualizado do usuário.

email: Um parâmetro do tipo string, Endereço de e-mail atualizado do usuário.

sexo: Um parâmetro do tipo string, Sexo do usuário (ex.: "Masculino" ou "Feminino").

data_nascimento: Um parâmetro do tipo string, Data de nascimento do usuário no formato ISO (YYYY-MM-DD).

cpf: Um parâmetro do tipo string, CPF atualizado do usuário (11 dígitos).

usuario: Um parâmetro do tipo int, ID do usuário responsável pela atualização (campo obrigatório).

Deletar

Para deletar um registro existente na tabela Usuários, envie uma solicitação contendo os seguintes parâmetros:

id_usuario: Um parâmetro do tipo int, ID único do usuário que será deletado (campo obrigatório).

usuario: Um parâmetro do tipo int, ID do usuário responsável pela exclusão (campo obrigatório).
