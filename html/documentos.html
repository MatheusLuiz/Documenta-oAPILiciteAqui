Listar

Não há necessidade de usar parâmetros para utilização desta API

Cadastrar

Para cadastrar um novo registro na tabela Documentos, envie uma solicitação contendo os seguintes parâmetros:

descricao: Um parâmetro do tipo string, Descrição do registro a ser cadastrado (campo obrigatório). Este campo deve conter uma breve descrição sobre o documento ou item a ser cadastrado.

usuario: Um parâmetro do tipo int, ID do usuário responsável pelo cadastro (campo obrigatório). Este campo identifica o usuário que está criando o registro.

Atualizar

Para atualizar um registro existente na tabela Documentos, envie uma solicitação contendo os seguintes parâmetros:

id_documento: Um parâmetro do tipo int, ID único do documento a ser atualizado (campo obrigatório). Este campo identifica o documento que será editado.

descricao: Um parâmetro do tipo string, Nova descrição do documento (campo obrigatório). Atualize este campo com a nova descrição do documento que está sendo modificado.

usuario: Um parâmetro do tipo int, ID do usuário responsável pela atualização (campo obrigatório). Este campo identifica o usuário que está fazendo a alteração no registro.

Deletar

Para deletar um registro da tabela Documentos, envie uma solicitação contendo os seguintes parâmetros:

id_documento: Um parâmetro do tipo int, ID único do documento a ser deletado (campo obrigatório). Este campo identifica o documento que será excluído.

usuario: Um parâmetro do tipo int, ID do usuário responsável pela exclusão (campo obrigatório). Este campo indica quem está realizando a ação de exclusão do documento.
