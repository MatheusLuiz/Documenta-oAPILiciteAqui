Listar Tipos de Documento

Para listar todos os tipos de documento disponíveis, utilize o seguinte endpoint:

  • GET /listar: Lista todos os tipos de documento.

Cadastrar Tipo de Documento

Para cadastrar um novo tipo de documento, envie uma solicitação POST contendo os seguintes parâmetros no corpo da requisição:

  • descricao: String - Descrição do tipo de documento (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pelo cadastro (campo obrigatório).

Atualizar Tipo de Documento

Para atualizar um registro existente na tabela Tipos de Documento, envie uma solicitação PUT contendo os seguintes parâmetros no corpo da requisição:

  • id_documento: Inteiro - ID único do documento a ser atualizado (campo obrigatório).
  • descricao: String - Nova descrição do tipo de documento (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pela atualização (campo obrigatório).

Deletar Tipo de Documento

Para deletar um registro existente na tabela Tipos de Documento, envie uma solicitação DELETE contendo os seguintes parâmetros no corpo da requisição:

  • id_documento: Inteiro - ID único do documento a ser deletado (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pela exclusão (campo obrigatório).
