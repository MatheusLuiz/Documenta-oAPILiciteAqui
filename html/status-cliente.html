Listar

Não há necessidade de usar parâmetros para utilização desta API

Cadastrar

Para cadastrar um novo registro na tabela Status Cliente, envie uma solicitação contendo os seguintes parâmetros:

descricao: Um parâmetro do tipo string, Descrição do status do cliente (campo obrigatório).

usuario: Um parâmetro do tipo int, ID do usuário responsável pelo cadastro (campo obrigatório).

Atualizar

Para atualizar um registro existente na tabela Status Cliente, envie uma solicitação contendo os seguintes parâmetros:

id_status: Um parâmetro do tipo int, ID único do status a ser atualizado (campo obrigatório).

descricao: Um parâmetro do tipo string, Nova descrição do status do cliente (campo obrigatório).

usuario: Um parâmetro do tipo int, ID do usuário responsável pela atualização (campo obrigatório).

Deletar

Para deletar um registro existente na tabela Status Cliente, envie uma solicitação contendo os seguintes parâmetros:

id_status: Um parâmetro do tipo int, ID único do status a ser deletado (campo obrigatório).

usuario: Um parâmetro do tipo int, ID do usuário responsável pela exclusão (campo obrigatório).
