Listar Status de Cliente

Para listar todos os status de cliente disponíveis, utilize o seguinte endpoint:

  • GET /listar: Lista todos os status de clientes.

Cadastrar Status de Cliente

Para cadastrar um novo status de cliente, envie uma solicitação POST contendo os seguintes parâmetros no corpo da requisição:

  • descricao: String - Descrição do status do cliente (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pelo cadastro (campo obrigatório).

Atualizar Status de Cliente

Para atualizar um registro existente na tabela Status Cliente, envie uma solicitação PUT contendo os seguintes parâmetros no corpo da requisição:

  • id_status: Inteiro - ID único do status a ser atualizado (campo obrigatório).
  • descricao: String - Nova descrição do status do cliente (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pela atualização (campo obrigatório).

Deletar Status de Cliente

Para deletar um registro existente na tabela Status Cliente, envie uma solicitação DELETE contendo os seguintes parâmetros no corpo da requisição:

  • id_status: Inteiro - ID único do status a ser deletado (campo obrigatório).
  • usuario: Inteiro - ID do usuário responsável pela exclusão (campo obrigatório).
